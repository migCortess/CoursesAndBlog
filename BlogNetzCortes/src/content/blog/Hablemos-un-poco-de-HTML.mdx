# Fundamentos de HTML

HTML, o HyperText Markup Language, es el lenguaje estándar para crear y diseñar páginas web. Aquí están los conceptos básicos que necesitas conocer:

## Estructura básica de un documento HTML

Un documento HTML típico tiene la siguiente estructura:

htmlCopiar código\<!DOCTYPE html>\<html lang="es">\<head>    \<meta charset="UTF-8">    \<meta name="viewport" content="width=device-width, initial-scale=1.0">    \<title>Título de la página\</title>\</head>\<body>    \<h1>Hola, mundo!\</h1>    \<p>Este es un párrafo de ejemplo.\</p>\</body>\</html>

1. \<!DOCTYPE html>: Esta declaración define el tipo de documento y la versión de HTML que se está utilizando (HTML5 en este caso).
2. \<html>: La etiqueta raíz que envuelve todo el contenido del documento.
3. \<head>: Contiene metadatos sobre el documento, como el conjunto de caracteres (\<meta charset="UTF-8">), la vista en dispositivos móviles (\<meta name="viewport" content="width=device-width, initial-scale=1.0">) y el título de la página (\<title>).
4. \<body>: Contiene el contenido visible de la página web, como encabezados (\<h1>), párrafos (\<p>), imágenes (\<img>), enlaces (\<a>), etc.

## Etiquetas HTML comunes

* htmlCopiar código\<h1>Encabezado 1\</h1>\<h2>Encabezado 2\</h2>
* htmlCopiar código\<p>Este es un párrafo.\</p>
* htmlCopiar código\<a href="https\://www\.ejemplo.com">Visitar ejemplo\</a>
* htmlCopiar código\<img src="imagen.jpg" alt="Descripción de la imagen">
* htmlCopiar código\<ul>  \<li>Elemento de lista 1\</li>  \<li>Elemento de lista 2\</li>\</ul>\<ol>  \<li>Primer elemento\</li>  \<li>Segundo elemento\</li>\</ol>
* htmlCopiar código\<table>  \<tr>    \<th>Encabezado 1\</th>    \<th>Encabezado 2\</th>  \</tr>  \<tr>    \<td>Dato 1\</td>    \<td>Dato 2\</td>  \</tr>\</table>

## Atributos de las etiquetas

Las etiquetas HTML pueden tener atributos que proporcionan información adicional sobre el elemento. Por ejemplo, la etiqueta \<a> puede tener un atributo href que especifica la URL a la que enlaza.

htmlCopiar código\<a href="https\://www\.ejemplo.com" target="\_blank">Visitar ejemplo\</a>

En este caso:

* href: La URL a la que apunta el enlace.
* target: Indica dónde abrir el enlace (en este caso, en una nueva pestaña con \_blank).

## Comentarios

Los comentarios en HTML se utilizan para dejar notas en el código y no se muestran en el navegador.

htmlCopiar código\<!-- Este es un comentario -->

## Conclusión

Estos son los fundamentos básicos de HTML. Comprender estas etiquetas y estructuras te permitirá crear páginas web sencillas y establecer una base sólida para aprender tecnologías web más avanzadas como CSS y JavaScript.
